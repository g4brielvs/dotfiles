# Dotfiles

This repository contains my dotfiles managed with [chezmoi](https://www.chezmoi.io/).

## Quick Start

To set up these dotfiles on a new machine, run:

```sh
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply g4brielvs
```
